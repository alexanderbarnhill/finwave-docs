## Web app or desktop client?

finwave is primarily a **web application** — that's where you submit encounters, review identifications, browse individuals, and run analyses. **Most users only ever need the web app.**

The **desktop client** is an optional companion for one specific workflow: bulk-importing photos directly from a local archive or cloud storage container that is too large to upload through the browser. If you don't have a backlog of thousands of historical photos to onboard, you can ignore the desktop client entirely.
